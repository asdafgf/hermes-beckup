## Name
hermes-agent

## Description
Hermes Agent is a part of a series focused on internal penetration testing, specifically within the Windows Active Directory environment. This skill aims to guide users through the critical aspects of security testing in a lab setting. 

## Body
### Overview
Hermes Agent features long-form, casual streams that provide a walkthrough of the new Hack The Box (HTB) lab environment, particularly aimed at internal penetration testing. The focus is on exploring and understanding the Windows Active Directory structure within those tests.

### Features
- **Active Directory Insights**: Learn the hierarchy of domains, trees, and forests in Active Directory and their security implications.
- **Live Interaction**: Engages with viewers live during the streams, asking for guidance and discussing strategies and techniques.
- **Practical Demonstrations**: Actual walkthroughs within the Hack The Box environment showcasing practical penetration testing methods.
- **Community Engagement**: Regularly responds to viewers in real time, creating a collaborative learning atmosphere.

### How to Get Started
1. **Tune into Streams**: The sessions are streamed live on Twitch under the channel [twitch.tv/jhamman010](https://twitch.tv/jhamman010). You can catch the ongoing series as well as past recordings.
2. **Engage with Content**: Participate in discussions during the live sessions or watch recorded content on YouTube for a more structured learning experience.
3. **Explore Active Directory**: Familiarize yourself with the concepts of domains, trees, and forests in Active Directory to enhance your understanding of penetration testing in this environment.
