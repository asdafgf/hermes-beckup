description: A step-by-step guide to enumerate Microsoft services using Graph Spy with access tokens.

body:
1. **Setup Graph Spy**: 
   - Visit the GitHub repository for Graph Spy.
   - Follow the installation instructions provided in the repository. This typically involves cloning the repository and installing any necessary dependencies.

2. **Launch the Tool**: 
   - After installation, run Graph Spy from your command line.
   - Create a new database for your tests to ensure no previous client access tokens are present.

3. **Access Tokens**: 
   - Familiarize yourself with how access tokens work, as they are critical for using Graph Spy effectively.

4. **Explore Features**:
   - Open Graph Spy in your web browser to view the user interface.
   - Observe the array of options available for enumeration and modification with access tokens.

5. **Perform Enumeration**:
   - Use the various command options to enumerate Microsoft services.
   - This includes reading emails, accessing Teams messages, and exploring SharePoint or OneDrive contents.

6. **Modify Information (if necessary)**:
   - Depending on your needs, leverage the modification capabilities within Graph Spy to adjust or interact with the data from Microsoft services.

7. **Consult Documentation**:
   - Utilize the wiki on the GitHub page to explore additional features and updates as they are added.
