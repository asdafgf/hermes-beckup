  ## What is Atomic Red Team?

  Atomic Red Team is a project that has been around for about five years. It's designed to help security professionals test or simulate attacker tradecraft in a controlled, benign environment. The project includes over 100 different atomics (scripts and commands) that can be used to perform various actions, such as privilege escalation, data exfiltration, and more.

  ## How Does Atomic Red Team Work?

  Atomic Red Team is integrated with many tools, including Splunk, which makes it a powerful resource for security professionals. You can submit your own atomics as pull requests (PRs) to the project, and if they are accepted, you'll receive a t-shirt or sticker as a thank you.

  ## Why Should I Use Atomic Red Team?

  Using Atomic Red Team can help organizations validate their detection engineering. By simulating attacker tradecraft in a benign environment, you can ensure that your systems are properly configured to detect and respond to potential threats.
