  ## Setup a Low Privilege User for Cybersecurity Testing

  ### Objective
  This guide will walk you through the process of setting up a low privilege user in an Entra ID or Microsoft 365 environment. This setup is essential for cybersecurity testing and penetration simulation.

  ### Prerequisites
  - Access to an Entra ID or Microsoft 365 tenant.
  - Administrative privileges to create users and manage groups.

  ### Steps

  1. **Access the Entra ID Admin Management Portal**
     - Open your web browser (e.g., Firefox) and navigate to `https://portal.azure.com`.
     - Log in with your global administrator credentials for the tenant.

  2. **Navigate to Users**
     - In the Azure portal, click on "Azure Active Directory" from the left-hand menu.
     - Click on "Users" under the "Manage" section.

  3. **Create a New User**
     - Click on the "+ New user" button at the top of the page.
     - Enter the following details:
       - **Display name**: Enter a name for the low privilege user (e.g., "LowPrivUser").
       - **User type**: Select "Standard user".
       - **Email address**: Provide an email address for the user (e.g., `lowprivuser@yourdomain.onmicrosoft.com`).
     - Set a password that is easy to remember but not autogenerated. Ensure it meets your organization's security policies.

  4. **Review and Create User**
     - Review all the details provided.
     - Click on "Create" to create the user.

  5. **Verify User Creation**
     - After creating the user, refresh the Users page to verify that the new low privilege user is listed.
     - Ensure the user has been created successfully and can be accessed from the Azure portal.

  ### Notes
  - This guide focuses on setting up a low privilege user for testing purposes. In a real-world scenario, ensure that all actions comply with your organization's cybersecurity policies and legal requirements.
  - The user created in this process should not have administrative privileges to prevent unauthorized access and escalation of privileges.

  ### Conclusion
  Setting up a low privilege user is a crucial step in cybersecurity testing. This guide provides a detailed process for creating such a user in an Entra ID or Microsoft 365 environment, ensuring that you can simulate real-world attack scenarios safely and effectively.
