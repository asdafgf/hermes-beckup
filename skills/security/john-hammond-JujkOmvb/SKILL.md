# skill: hack-windows-recall

## Description
Learn how to exploit the privacy vulnerabilities of the Windows Recall feature.

## Body
### Step-by-Step Guide:

1. **Understand Windows Recall**: 
   - Familiarize yourself with Windows Recall, an AI tool developed by Microsoft that takes continuous snapshots of your activities on your computer.

2. **Set Up a Testing Environment**: 
   - Use an Azure Cloud virtual machine running the latest preview version of Windows 11 (24 H2).

3. **Access Windows Recall**: 
   - Locate the Windows Recall icon in the system tray of your desktop.

4. **Interact with Your Computer**:
   - Perform various tasks on the computer to generate data that Recall will store for retrieval.

5. **Open Windows Recall**:
   - You can open Recall by clicking the icon or pressing the Windows key + J.

6. **Explore Your Data**:
   - Use Recall’s features to search through your captured history, including text and images of your activities.

7. **Analyze Privacy Risks**:
   - Review the types of information collected by Recall and identify potential privacy issues, particularly sensitive or personal data.

8. **Demonstrate the Vulnerability**:
   - Showcase an example where Recall retrieves sensitive information that users may not want readily available.

9. **Discuss the Implications**:
   - Engage in conversation about the potential consequences of this feature on user privacy and security.

10. **Research Further**:
   - Stay updated on any patches or changes Microsoft may implement to address these security concerns.
