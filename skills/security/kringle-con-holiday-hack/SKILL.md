body: |
  Welcome to the KringleCon 2021 Holiday Hack Challenge! This educational and interactive event from SANS provides participants with various objectives designed to enhance their cybersecurity skills. 

  In this session, we dive into Objective 8: "Kerber Roasting on an Open Fire." The main task is to obtain the secret slave research document from a host on the Elf University domain. To start, participants must register as a student on the Elf U portal and find Eve Snowshoes in Santa's office for hints. 

  Eve Snowshoes provides a new challenge involving the "Fail2Ban" application, which is useful for monitoring logs and blocking malicious IP addresses. The task requires configuring Fail2Ban to detect and block IP addresses that generate 10 or more failure messages within an hour. 

  The main objectives are:
  - Obtain the secret research document.
  - Configure Fail2Ban to automate the blocking of bad IP addresses.
  - Understand how to manage the naughty list, using commands to add, remove, and check which IPs are currently blocked. 

  Throughout the challenge, participants learn about Kerberos and Active Directory permissions while engaging in a fun and festive environment. This is a prime opportunity to sharpen your skills in cybersecurity and network monitoring!
