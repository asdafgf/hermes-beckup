name: take-ownership

description: Take ownership of a file or folder to gain administrative privileges.

body:
1. Open File Explorer.
2. Navigate to the file or folder you want to modify.
3. Right-click on the file or folder and select "Properties".
4. Click on the "Security" tab.
5. Click on "Edit" next to the owner field.
6. If prompted, enter your administrator credentials.
7. In the "Select User or Group" dialog box, type your username (e.g., `John`) and click "Check Names".
8. Click "OK" to apply the changes.
9. Back in the Security tab, check the box next to "Replace owner on subcontainers and objects".
10. Click "Apply" and then "OK".

category: security
