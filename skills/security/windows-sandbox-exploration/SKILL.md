  ## Introduction to Windows Sandbox

  In March of 2025, the Ito Cyber and Intelligence Incorporation released a writeup titled "Hack the Sandbox," which detailed how the threat actor Mirrorface uses Windows Sandbox and Visual Studio Code in their attack campaigns targeting organizations in Japan. This video aims to showcase Windows Sandbox for testing files or experimenting with potentially malicious files in a safe isolated environment.

  ## Setting Up Windows Sandbox

  If you aren't familiar with Windows Sandbox, it is a virtual machine that provides an isolated environment from your host computer. To enable Windows Sandbox:

  1. Search for "Windows features" on the start menu.
  2. Scroll down and find "Windows Sandbox."
  3. Click to enable it and follow the prompts to reboot your computer.

  ## Using Windows Sandbox

  Once enabled, you can use Windows Sandbox to test files or experiment with potentially malicious files in a safe isolated environment. Here’s how:

  - Open the Start menu.
  - Search for "Windows Sandbox."
  - Click on it to open the sandboxed environment.

  ## Practical Applications

  If you are a redteamer, penetration tester, or anyone doing offensive security, learning about Windows Sandbox can be very worthwhile. It allows you to emulate cyber crime and understand what threat actors have up their sleeve.

  ## Conclusion

  Windows Sandbox is a powerful tool for cybersecurity professionals. By using it, you can test files in a safe isolated environment without risking your host computer. Whether you are testing malware or experimenting with new technologies, Windows Sandbox provides the perfect solution.
