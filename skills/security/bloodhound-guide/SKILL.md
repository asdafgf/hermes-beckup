# Step-by-Step Guide to Using BloodHound in Active Directory

## Step 1: Setup Your Environment
1. Ensure you have a Kali Linux virtual machine set up.
2. Confirm that you have a domain controller running within your lab environment.
3. Optionally, set up a management client and a workstation.

## Step 2: Access the Kali Linux Terminal
1. Open your terminal by pressing `Ctrl + Alt + T`.

## Step 3: Change Directory
1. Navigate to your Active Directory folder using the command:
   ```bash
   cd /path/to/your/active_directory_directory
   ```
   Replace `/path/to/your/active_directory_directory` with your actual directory path.

## Step 4: Prepare Your Working Directory
1. Create a new directory for BloodHound:
   ```bash
   mkdir bloodhound
   cd bloodhound
   ```

## Step 5: Install BloodHound
1. If you haven’t installed BloodHound, you’ll need to do this. You can find installation instructions on the official GitHub repository.
2. Make sure all dependencies are resolved.

## Step 6: Collect Data
1. Use `SharpHound`, the data collector for BloodHound, to gather information from the domain. This typically involves running a PowerShell script on a compromised system.
2. Ensure you execute the data collector based on the proper permissions to gather meaningful data.

## Step 7: Load Data into BloodHound
1. Once the data collection is complete, open BloodHound.
2. Import the JSON files generated by `SharpHound` into BloodHound.

## Step 8: Analyze the Data
1. Utilize BloodHound’s interface to analyze relationships and paths within the Active Directory.
2. Look for potential attack paths and misconfigurations that could be exploited.

## Step 9: Document Findings
1. As you discover potential vulnerabilities, remember to document your findings for future reference.
2. Use a README file or markdown notes within your directory to capture your insights.

## Conclusion
Follow these steps to effectively utilize BloodHound in your Active Directory assessments. Understanding the relationships and permissions within AD can significantly enhance your security posture.
