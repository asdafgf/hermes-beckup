# Step-by-Step Guide to Using Graph Spy

## Step 1: Installation
1. **Access GitHub Repository**: Go to the GitHub page where Graph Spy is hosted.
2. **Download the Tool**: Navigate to the releases section and download the latest version.
3. **Follow Installation Instructions**: Refer to the 'Installation' page on the GitHub repository for specific setup steps.

## Step 2: Setting Up Your Environment
1. **Create a Secure Environment**: Start by setting up a clean environment or database for testing, avoiding any prior access tokens.
2. **Launch Graph Spy**: Open your terminal and run Graph Spy using the provided command. It should launch the application in your default browser.

## Step 3: Understand the Basics
1. **Access Tokens**: Familiarize yourself with how access tokens work within Azure and Graph API.
2. **Interface Overview**: Upon launch, observe the user interface which displays various options and features available for enumeration.

## Step 4: Performing Enumeration
1. **Identify Features**: At the top of the interface, view the available enumeration options such as reading emails, accessing Teams messages, and browsing SharePoint or OneDrive.
2. **Select an Action**: Choose the desired action to execute based on the access tokens that you have.

## Step 5: API Calls
1. **Use Access Tokens**: After obtaining access tokens—bear in mind these do not require a username or password—you can send API calls to Microsoft services.
2. **Run Commands**: Execute commands directly from the Graph Spy interface to interact with the chosen Microsoft services.

## Step 6: Explore Further
1. **Utilize Wiki Resources**: Check out the wiki on the GitHub repository for detailed explanations of features and commands.
2. **Stay Updated**: Keep track of any updates or new features added to the tool by following its repository.

## Step 7: Practice and Experiment
1. **Test Features**: Try out various enumeration and modification features to become proficient with Graph Spy.
2. **Share Your Experience**: Engage with the community by sharing your findings or any challenges faced while using the tool.

By following this guide, users will be equipped to utilize Graph Spy effectively for their cybersecurity engagements.
