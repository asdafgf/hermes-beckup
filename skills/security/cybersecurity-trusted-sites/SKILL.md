  ## Introduction

  This website catalogs and archives a list of other websites that could be used and abused by hackers. These include both good and bad actors, such as ethical hackers, penetration testers (red teamers), and cyber criminals.

  ## Living off Trusted Sites

  "Living off trusted sites" is a technique where hackers use legitimate domains and websites to evade detection when performing malicious activities like fishing, exfiltration, or command and control. These sites are not typically on deny lists or filtered by internet filters.

  ## Example: File Transfer

  Let's take an example of file transfer. A website might offer options for downloading things, uploading malicious files, or using it as a link for data exfiltration. This could be done using familiar services like GitHub or raw.githubusercontent.com.

  ## Using Software-Focused Services

  Many software-focused services related to coding and development can also be used for malicious purposes. GitHub is one of the top options for hosting files or code, making it a popular choice for command and control communications.
