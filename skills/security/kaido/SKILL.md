body: |
  Kaido is an innovative web application security testing tool aimed at helping security professionals and enthusiasts conduct efficient audits of web applications. With similarities to tools like Burp Suite and OWASP ZAP, Kaido brings modern features that enhance usability and extendability.

  **Key Features of Kaido:**
  - **Built with Rust**: Kaido is developed in Rust, providing performance that is both fast and efficient.
  - **Extensible Functionality**: The tool is easy to extend, allowing users to add new capabilities as needed.
  - **Efficient Workspace Management**: Users can manage multiple projects with a clean and structured workspace.
  - **Cross-Platform Support**: Kaido runs on various operating systems, including Windows, Linux, and macOS, making it accessible to a wider audience.
  - **Easy Installation and Setup**: Users can quickly download and install Kaido via an MSI file, getting it up and running effortlessly.
  
  **Getting Started with Kaido:**
  1. Download the desktop application or use the command line interface to host it remotely.
  2. Create an account and log in to access Kaido's features.
  3. Set up a new project to begin your security auditing processes.
  4. Start the proxy server on port 8080, similar to other tools like Burp Suite.
  5. Explore Kaido's intuitive dashboard and get acquainted with its functionalities.

  Kaido is a powerful addition to the security testing toolkit that brings modern capabilities to web application auditing, making it easier to identify and mitigate vulnerabilities effectively.
